# Azure DevOps Essentials

**Everything you need to be productive on Microsoft Azure — before touching AKS.**

Microsoft built Azure to extend Windows Server, Active Directory, and enterprise IT into the cloud. This track covers the core Azure services every DevOps engineer needs: identity (Entra ID), networking, compute, containers, serverless, secrets, observability, and CI/CD.
