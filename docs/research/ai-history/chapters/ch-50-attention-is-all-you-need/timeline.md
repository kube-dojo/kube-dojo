# Timeline: Chapter 50

- **2014:** Sutskever, Vinyals, and Le publish an LSTM sequence-to-sequence system for machine translation, encoding input into a fixed-dimensional vector and decoding output with another LSTM.
- **2014/2015:** Bahdanau, Cho, and Bengio introduce neural machine translation with learned soft alignment, addressing the fixed-vector bottleneck while still working in a recurrent encoder-decoder setting.
- **June 2017:** Vaswani et al. submit "Attention Is All You Need" to arXiv, proposing the Transformer architecture.
- **December 2017:** The paper appears at NeurIPS 2017. Its reported machine-translation experiments use one machine with 8 NVIDIA P100 GPUs.
- **Later Part 8:** BERT, GPT-2, Hugging Face, scaling laws, OpenAI/Microsoft infrastructure, RLHF, and diffusion are downstream chapters, not Chapter 50 content.
