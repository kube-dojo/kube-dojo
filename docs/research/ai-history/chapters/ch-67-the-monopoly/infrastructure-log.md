# Infrastructure Log: Chapter 67 - The Monopoly

## Systems To Track

- **Accelerators:** NVIDIA Hopper/Grace Hopper/data-center platforms, plus
  hyperscaler custom chips such as AWS Trainium and Google TPUs.
- **Cloud capacity:** Azure, AWS, and Google Cloud as training/inference surfaces,
  not interchangeable commodity hosting.
- **Partnership contracts:** investment, cloud-provider status, IP licenses,
  revenue share, first-ship clauses, API exclusivity, and right-of-first-refusal
  changes.
- **Distribution channels:** Azure OpenAI-style enterprise channels, Amazon
  Bedrock, Google Cloud integrations, and first-party products.
- **Regulatory visibility:** FTC 6(b) orders and staff report; joint
  competition-authority statement.
- **Evolving terms:** 2019 Microsoft/OpenAI exclusivity differs from 2025, Feb.
  2026, and Apr. 2026 statements.

## Metrics And Claims To Keep Source-Bound

- NVIDIA FY2024 Data Center revenue: $47.5B, up 217%; large cloud providers more
  than half of Q4 FY2024 Data Center revenue; about 40% of FY2024 Data Center
  revenue estimated for AI inference.
- Microsoft/OpenAI 2019: $1B investment; Azure AI supercomputing; Microsoft as
  exclusive cloud provider and preferred commercialization partner.
- Microsoft/OpenAI 2025-2026: exclusive IP/Azure API terms in Oct. 2025/Feb.
  2026; Apr. 27, 2026 primary cloud / all-cloud serving / non-exclusive license.
- Amazon/Anthropic: $4B September 2023 investment, additional $4B in 2024, AWS
  primary cloud/training partner, Trainium for largest foundation models.
- Google/Anthropic: Google Cloud provider/preferred provider; GPU/TPU clusters;
  TPU v5e inference partnership.
- FTC report public-dollar table: $13.75B Microsoft-OpenAI, $8B
  Amazon-Anthropic, $2.55B Google-Anthropic, based on public reporting and
  press releases.
