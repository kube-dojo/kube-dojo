# Timeline: Chapter 67 - The Monopoly

- **2019-07-22:** OpenAI announces Microsoft's $1B investment, Azure
  supercomputing partnership, and Microsoft as exclusive cloud provider and
  preferred commercialization partner.
- **2023-02-03:** Anthropic announces Google Cloud partnership for GPU/TPU
  training, scaling, and deployment infrastructure.
- **2023-09:** Amazon and Anthropic announce strategic collaboration, with AWS as
  Anthropic's primary cloud provider and Amazon investment.
- **2024-01-25:** FTC launches 6(b) inquiry into generative-AI investments and
  partnerships involving Alphabet, Amazon, Anthropic, Microsoft, and OpenAI.
- **2024-07-23:** US, EU, and UK competition authorities publish joint statement
  on competition in generative AI foundation models and AI products.
- **2024-11-22:** Amazon announces deeper Anthropic collaboration, additional
  $4B investment, AWS as primary training partner, and Trainium use.
- **2025-01:** FTC staff report on cloud service provider / AI developer
  partnerships describes Microsoft-OpenAI, Amazon-Anthropic, and
  Alphabet-Anthropic as the study's largest such partnerships.
- **2025-10-28:** Microsoft announces revised OpenAI agreement, retaining
  exclusive IP rights / Azure API exclusivity while allowing some third-party and
  non-API flexibility.
- **2026-02-27:** Microsoft/OpenAI joint statement clarifies Azure exclusivity
  for stateless OpenAI APIs and first-party products.
- **2026-04-27:** OpenAI and Microsoft announce amended agreement: Microsoft
  remains primary cloud partner, but OpenAI can serve products across any cloud
  and Microsoft's license becomes non-exclusive.
