# Scene Sketches: Chapter 32

## Scene 1: The Brittle Rules
- **Action:** A linguist tries to write a rule for every way an American might pronounce "tomato," resulting in combinatorial explosion.

## Scene 2: The Benchmark
- **Action:** DARPA stops funding "promising theories" and demands systems run against a hidden test tape.

## Scene 3: The Statistical Triumph
- **Action:** Jelinek's team feeds massive amounts of text into Markov models, letting the machine count probabilities rather than learn grammar.
