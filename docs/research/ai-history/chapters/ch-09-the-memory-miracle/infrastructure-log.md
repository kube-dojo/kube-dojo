# Infrastructure Log: Chapter 9

## Technical Metrics & Constraints
- **Williams Tubes (Late 1940s):**
  - Reliability: Extreme volatility. Erased by static, passing streetcars, or power loss.
- **Magnetic Core Memory (1950s):**
  - Reliability: Non-volatile. Data remained indefinitely.
  - Scale: Woven grids of ferrite cores. It was the standard memory infrastructure from 1955 until the silicon RAM chip in the 1970s.
