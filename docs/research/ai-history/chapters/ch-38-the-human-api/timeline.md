# Timeline: Chapter 38 - The Human API

- **2001-03-19:** Priority date for the "Hybrid machine/human computing arrangement" patent later published as US7197459B1. The patent describes a system in which a computer decomposes tasks into subtasks for human performance and uses an API to request those performances.
- **2001-10-12:** Filing date for US7197459B1. Inventors listed: Venky Harinarayan, Anand Rajaraman, and Anand Ranganathan; original assignee listed as Amazon Technologies Inc.
- **Early 2000s:** Secondary sources report that Amazon faced duplicate product listings in its growing catalog and that automated duplicate removal proved difficult. This is the setup for the catalog scene, but it remains secondary-anchored.
- **2004:** Luis von Ahn and Laura Dabbish publish "Labeling Images with a Computer Game" at CHI, an important pre-MTurk human-computation/image-labeling boundary source.
- **2005-11-02:** AWS publicly announces Amazon Mechanical Turk, describing a web-services API for integrating "Artificial Artificial Intelligence" into computer processing.
- **2006-09-27:** Bezos gives an MIT Emerging Technologies keynote reported by Computerworld as a look at the "hidden Amazon"; MTurk appears beside S3 and EC2 as one of Amazon's developer web services.
- **2007-03-25:** Jason Pontin's New York Times article "Artificial Intelligence, With Help From the Humans" appears. It is frequently cited for Bezos's public explanation, but the text was not directly accessible in this environment.
- **2007-03-27:** US7197459B1 is granted/published.
- **2008-10:** Snow, O'Connor, Jurafsky, and Ng publish "Cheap and Fast - But is it Good?" at EMNLP, evaluating MTurk for five NLP annotation tasks.
- **2009-06:** Deng et al. publish "ImageNet: A Large-Scale Hierarchical Image Database" at CVPR, reporting 12 subtrees, 5,247 synsets, 3.2 million images, and a construction pipeline using AMT.
- **2018-04:** Hara et al. publish a CHI task-level wage analysis of AMT workers, useful as a later boundary source on what the API abstraction hid.
