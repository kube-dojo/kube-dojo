# Scene Sketches: Chapter 8

## Scene 1: The Plugs and Cables
- **Setting:** Moore School of Electrical Engineering, UPenn (1945).
- **Action:** Women mathematicians ("computers") physically routing cables to calculate artillery tables.

## Scene 2: The EDVAC Report
- **Action:** Von Neumann reads McCulloch, Pitts, and Turing, and synthesizes a physical machine that uses logic gates and a central memory bank.

## Scene 3: The Birth of Software
- **Action:** The paradigm shift. You don't build a chess-playing machine; you build a general-purpose machine and write a chess-playing text file.
