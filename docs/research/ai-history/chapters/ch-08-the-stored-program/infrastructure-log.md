# Infrastructure Log: Chapter 8

## Technical Metrics & Constraints
- **ENIAC (1945):**
  - Architecture: Fixed physical pathways. Programming took days of manual labor.
- **Von Neumann Architecture:**
  - Architecture: Central Processing Unit (CPU) executing sequential instructions fetched from a unified Memory containing both data and programs.
  - Constraint: It required memory infrastructure that was fast and reliable enough to fetch instructions without crashing.
