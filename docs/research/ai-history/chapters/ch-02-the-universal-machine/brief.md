# Brief: Chapter 2 - The Universal Machine

## Thesis
The concept of software was born not from an engineering breakthrough, but from a mathematical puzzle. Alan Turing's theoretical "Universal Machine" proved that hardware did not need to be specialized for every task; a single machine could execute any computable logic if given the right instructions.

## Scope
- IN SCOPE: Hilbert's *Entscheidungsproblem*, Alan Turing's 1936 paper, the concept of the Turing Machine (tape, read/write head, states).
- OUT OF SCOPE: Turing's codebreaking work at Bletchley Park (belongs to later chapters).

## Scenes Outline
1. **Hilbert's Challenge (1928):** David Hilbert asks if mathematics can be automated.
2. **The Meadow at King's College (1936):** Alan Turing conceptualizes a machine that operates on an infinite paper tape.
3. **The Ultimate Abstraction:** Turing proves that a single "Universal Machine" can simulate any other machine, birthing the concept of software.
