# Infrastructure Log: Chapter 56

## Technical Metrics & Constraints
- **The Azure Supercomputer (2020 announcement):**
  - Specs: A single system with more than 285,000 CPU cores, 10,000 GPUs, and 400 gigabits per second of network connectivity for each GPU server.
  - Constraint: Connecting 10,000 GPUs requires highly specialized networking (InfiniBand) so the GPUs don't sit idle waiting for data.
