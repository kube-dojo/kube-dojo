# Timeline: Chapter 4 — The Statistical Roots

Dates are given in the New Style (Gregorian) calendar except where noted as Old Style (Julian). Russia did not adopt the Gregorian calendar until 1918, so events before 1918 are most often cited Old Style in the primary record; this timeline gives O.S. with N.S. equivalents where the conversion is established.

## Key Calendar Note

Russia used the Julian calendar until February 1918. The 1913 Onegin lecture date "23 January 1913" in the published primary source (Markov 1913, Link translation, p.591 footnote 1) is Old Style; the New Style equivalent is February 5, 1913. Use Old Style in prose because that is how the lecture was dated in the original record; flag the conversion only if narrative clarity demands it.
