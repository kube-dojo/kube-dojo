# Brief: Chapter 42 - CUDA

## Thesis
NVIDIA realized that the graphics hack was a massive new market. By releasing CUDA (Compute Unified Device Architecture) in 2006, they provided a standardized, C-like programming interface that abstracted away the graphics pipeline, transforming the GPU from a gaming accessory into a general-purpose supercomputer.
