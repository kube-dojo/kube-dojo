## Чому цей модуль важливий

Іспити з множинним вибором потребують іншої підготовки, ніж практичні іспити. Вас перевіряють не на здатності робити — а на здатності розуміти та розпізнавати правильні відповіді.

Цей модуль навчить вас ефективно готуватися до KCNA.

## Стратегії множинного вибору

### Стратегія 1: Виключення

Спочатку видаліть очевидно неправильні відповіді:

```
Питання: Що НЕ є компонентом площини управління?
A) kube-apiserver    ← Компонент площини управління
B) etcd              ← Компонент площини управління
C) kubelet           ← Компонент ВУЗЛА! Відрізняється.
D) kube-scheduler    ← Компонент площини управління

Відповідь: C (kubelet працює на вузлах, не на площині управління)
```

### Стратегія 2: Слідкуйте за абсолютами

Відповіді зі словами "завжди", "ніколи", "всі", "жоден" часто неправильні:

```
Питання: Поди завжди містять кілька контейнерів.
→ ХИБНО (Поди МОЖУТЬ мати один контейнер)

Питання: Services ніколи не використовують ClusterIP.
→ ХИБНО (ClusterIP — це тип за замовчуванням!)
```

### Стратегія 3: Шукайте "найкращу" відповідь

Іноді кілька відповідей здаються правильними — виберіть НАЙКРАЩУ:

```
Питання: Яке основне призначення Deployment?
A) Запуск одного Пода              ← Правда, але обмежено
B) Декларативне керування ReplicaSets  ← НАЙКРАЩЕ: охоплює повну мету
C) Створення Services               ← Неправильно
D) Зберігання конфігурації           ← Неправильно
```

### Стратегія 4: Позначте та поверніться

Не зависайте:
1. Відповідайте впевнено, якщо знаєте
2. Зробіть найкращу здогадку, якщо не впевнені
3. Позначте для перегляду
4. Поверніться зі свіжим поглядом

### Математика часу

- 90 хвилин / 60 питань = 1,5 хвилини на питання
- Прагніть 1 хвилину на легкі питання
- Зберігайте час для складніших питань
- Ніколи не залишайте питання без відповіді

## Чи знали ви?

- **Розпізнавання перемагає пригадування** - Дослідження показують, що люди можуть розпізнати правильні відповіді, навіть коли не можуть пригадати їх з пам'яті. Множинний вибір використовує цю перевагу.

- **Інтервальне повторення працює** - Повторення матеріалу через зростаючі інтервали (1 день, 3 дні, 7 днів) значно покращує запам'ятовування.

- **Сон закріплює навчання** - Ваш мозок обробляє та зберігає інформацію під час сну. Не зубріть напередодні.

- **Тривога перед тестом — це нормально** - Навіть досвідчені фахівці нервують. Глибоке дихання та позитивний внутрішній діалог допомагають.

## Підсумок

**Готуйтеся інакше для множинного вибору**:
- Зосередьтеся на концепціях, а не командах
- Розумійте "чому", а не тільки "що"
