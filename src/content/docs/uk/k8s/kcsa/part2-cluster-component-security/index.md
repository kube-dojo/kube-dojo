Ця частина зосереджена на захисті площини управління, вузлів, мережі кластера та PKI — фундаменті перед контролем робочих навантажень.

## Модулі

| Модуль | Тема | Складність | Час |
|--------|------|------------|-----|
| [2.1](module-2.1-control-plane-security/) | Безпека площини управління | Середня | 20-30 хв |
| [2.2](module-2.2-node-security/) | Безпека вузлів | Середня | 20-30 хв |
| [2.3](module-2.3-network-security/) | Мережева безпека кластера | Середня | 20-30 хв |
| [2.4](module-2.4-pki-certificates/) | PKI та сертифікати | Середня | 20-30 хв |

## Що ви дізнаєтесь

- Зміцнення доступу до API server та etcd
- Захист на рівні вузлів (kubelet, ОС, середовище виконання)
- Мережеві контролі та потоки трафіку всередині кластера
- Основи PKI для сертифікатів Kubernetes
